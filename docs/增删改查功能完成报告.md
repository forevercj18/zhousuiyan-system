## 技术实现

### 权限控制
所有增删改查操作都添加了权限装饰器：
- `@require_permission('module', 'create')` - 创建权限
- `@require_permission('module', 'update')` - 更新权限
- `@require_permission('module', 'delete')` - 删除权限
- `@require_permission('module', 'view')` - 查看权限

### 业务规则
1. **订单删除**：只有待处理状态的订单可以删除
2. **采购单删除**：只有草稿状态的采购单可以删除
3. **部件删除**：只有库存为0的部件可以删除（软删除）
4. **SKU删除**：软删除，设置is_active=False
5. **部件出库**：自动检查库存是否充足

### 用户体验
1. **确认对话框**：所有删除操作都有确认提示
2. **成功提示**：操作成功后显示消息提示
3. **错误处理**：捕获异常并显示友好的错误消息
4. **模态框**：新建和编辑使用模态框，避免页面跳转
