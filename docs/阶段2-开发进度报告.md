```python
    2. 查询该期间内有冲突的订单
    3. 统计已占用数量
    4. 可用数量 = 总库存 - 已占用
    """
```

**示例：**
- SKU总库存：5套
- 活动日期：2026-03-15
- 发货提前：2天（3月13日发货）
- 回收延后：1天（3月17日回收）
- 缓冲天数：1天
- 占用期间：3月13日 - 3月18日
- 该期间已有2个订单占用2套
- 可用数量：5 - 2 = 3套
