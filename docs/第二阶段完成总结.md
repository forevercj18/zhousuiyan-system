- 占用期间：3月13日 - 3月18日
- 该期间已有2个订单占用2套
- 可用数量：5 - 2 = 3套

## 📊 系统特性

### 技术亮点

1. **事务处理**：所有写操作都使用`@transaction.atomic`保证数据一致性
2. **权限控制**：基于角色的访问控制（RBAC），支持模块级和操作级权限
3. **操作日志**：中间件自动记录用户操作，无需手动调用
4. **库存算法**：考虑发货提前、回收延后、缓冲天数的完整库存校验
5. **服务层设计**：业务逻辑与视图分离，便于测试和维护
6. **数据库优化**：合理的索引设计，提升查询性能
