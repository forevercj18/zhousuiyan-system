```bash

```

访问：http://localhost:8000

### 方式二：本地开发

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 数据库迁移
python manage.py migrate

# 3. 创建管理员
python scripts/create_superuser.py

# 4. 启动服务器
python manage.py runserver 8000
```

访问：http://localhost:8000

## 🔒 安全提示

1. **修改默认密码**：首次登录后立即修改admin密码
2. **更换SECRET_KEY**：生产环境必须更换Django的SECRET_KEY
3. **关闭DEBUG**：生产环境设置DEBUG=False
4. **配置ALLOWED_HOSTS**：限制允许访问的域名
5. **数据备份**：定期备份db.sqlite3数据库文件
6. **HTTPS部署**：生产环境使用HTTPS协议
7. **防火墙配置**：限制端口访问权限
